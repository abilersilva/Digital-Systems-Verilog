# Digital Systems in Verilog ⚡

Coleção de módulos em **Verilog/SystemVerilog** desenvolvidos para estudo e prática de sistemas digitais.  
Inclui conversores de código, máquina de estados, calculadora e máquina de vendas, todos com **testbenches** para simulação.

## 🚀 Projetos incluídos
- **Conversor de códigos**: Implementa diferentes codificações (BCD 8421, Excesso 3, Gray, etc.).
- **Máquina de estados**: FSM com entradas I e J, codificação one-hot e saída monitorada.
- **Calculadora de 8 bits**: Suporta adição, subtração, multiplicação e divisão com detecção de erro (divisão por zero).
- **Máquina de vendas**: Controle de saldo, preços e troco, com suporte a moedas de 25, 50 e 100 centavos.

## 📦 Tecnologias
- Verilog / SystemVerilog  
- Aldec Riviera PRO 2023.04  
- EPWave para análise de ondas
  
📌 Funcionalidades principais
- Conversão entre diferentes códigos digitais
- Simulação de máquina de estados com transições condicionais
- Operações aritméticas básicas com tratamento de erros
- Sistema de vendas com controle de saldo e troco
